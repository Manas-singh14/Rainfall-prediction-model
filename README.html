Overview
About the Project

District-wise monsoon intelligence for all of UP

Historical daily weather records for every Uttar Pradesh district were collected from ERA5 (ECMWF) and NASA POWER APIs. A supervised multi-horizon dataset was constructed, then dual XGBoost models — one regressor for millimetre amounts, one classifier for rain/no-rain probability — were trained and evaluated per forecast day. The same pipeline can ingest live Open-Meteo data to produce real-time 7-day outlooks.

How It Works

01

Data Ingestion

ERA5 reanalysis + NASA POWER daily records per district. Kelvin → Celsius conversion, 99th-percentile rain capping.

02

Feature Engineering

Lag-1/2/3/7/14 rainfall, rolling 3/7/14/30-day stats, dewpoint depression, moisture flux, cyclic month/DOY encoding.

03

Supervised Construction

Multi-horizon pivot: each (district, date) row is replicated 7×, one per target day. Target-date weather features are shifted accordingly.

04

Dual XGBoost Training

Regressor predicts log1p(mm); Classifier predicts rain probability. Evaluated per horizon day on a chronological train/test split.

05

Live Forecast

Geocodes any UP place name, fetches Open-Meteo hourly, builds feature rows, runs inference, and saves a dual-axis prediction chart.

06

EDA & Risk Scoring

District, monthly, yearly risk scores (0–100) via percentile composites. Heatmaps, scatter plots, and tabular CSVs saved to output directory.

Usage
Command-line interface

Quick Start

# Train the model on your CSV

$
python rainfall_7day_forecast.py train \ --data up_daily_weather_dataset.csv \ --output-dir models_7day \ --train-end-year 2022

# Predict from CSV data for a specific district

$
python rainfall_7day_forecast.py predict \ --district Lucknow \ --as-of-date 2023-06-15 \ --plot-path lucknow_forecast.png

# Live forecast via Open-Meteo (no CSV needed)

$
python rainfall_7day_forecast.py live-predict \ --place Varanasi \ --rain-probability-threshold 0.4

# Exploratory Data Analysis + risk tables

$
python rainfall_7day_forecast.py eda \ --data up_daily_weather_dataset.csv \ --output-dir eda_outputs

XGBoost Configuration

rainfall_7day_forecast.py — make_models()
def make_models(random_state: int) -> tuple[XGBRegressor, XGBClassifier]:
    regressor = XGBRegressor(
        objective    = "reg:squarederror",
        n_estimators = 450,
        max_depth    = 7,
        learning_rate= 0.04,
        subsample    = 0.8,
        reg_lambda   = 2.0,
    )
    classifier = XGBClassifier(
        n_estimators = 350,
        max_depth    = 6,
        learning_rate= 0.05,
        eval_metric  = ["logloss", "error"],
    )
    return regressor, classifier

# Targets
# regressor  → log1p(rainfall_mm)   [expm1 on output]
# classifier → P(rainfall > 0.1 mm) [threshold = 0.5]
